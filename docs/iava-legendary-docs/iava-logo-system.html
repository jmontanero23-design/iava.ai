Logo Symbolism

The Prediction Arc

The curved line at the top represents iAVA's predictive AI capabilities - the ability to forecast market movements before they happen.

The Neural Network

The connected nodes represent the AI neural network that powers iAVA - intelligence distributed across multiple points, working together.

The Tree Structure

From one insight, branches grow into multiple opportunities. Like a decision tree, each node expands possibilities.

The 7 Nodes

Representing completeness and perfection. The central node is AVA Mind, branching to the core AI features.

Usage Guidelines

Do's ✓

  • Use the gradient version on dark backgrounds
  • Maintain minimum clear space (1x logo width)
  • Use white version on gradient backgrounds
  • Scale proportionally
  • Use animated version for loading states

Don'ts ✗

  • Stretch or distort the logo
  • Change the gradient direction
  • Use on busy backgrounds
  • Add effects like drop shadows
  • Rearrange the nodes
